Odoo 14 ITU Sequence by YEAR

Módulo que adapta la secuencia de factyuras a código/Año/Numero Factura:

Module that adapts the sequence of Invoices to code/Year/Number.

Instalación:
Este módulo requiere de los módulos:
  • account
  • itu_log
Instalation:
That module requiere this extra modules:
  • account
  • itu_log
Configuración:
  • Crea un checkbox en los Diarios de contabilidad (account.journal) para activar o desactivar la secuencia usando solo el año.
Configuration:
  • Create a checkbox in the accounting journals (account.journal) to enable or disable the sequence using only the year.
Uso:

Al Crear una factura se comprueba si el cliente es anónimo y si tiene DNI/CIF. En ese caso se crea una factura simplificada en vez de una normal.

Usage:

When creating an invoice it is checked if the customer is anonymous and if he has a DNI/CIF. In this case a simplified invoice is created instead of a normal one.

Known issues:

El modulo da error al crear una factura simplificada si no tienes configurada la secuencia de facturas "normales" o si toca regenerarla, por ejemplo al cambiar de mes o año. Puedes solucionar este inconveniente simplemente generando una factura "normal".

The module gives error when creating a simplified invoice if you have not configured the sequence of "normal" invoices or if you have to regenerate it, for example when changing month or year. You can solve this problem by simply generating a "normal" invoice.
