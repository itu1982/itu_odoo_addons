Odoo 14 ITU LOG

Modulo que permite enviar mensajes al archivo log, siempre y cuando estemos en modo Debug.

Module that allows sending messages to the log file, as long as we are in Debug mode.

 

Uso:

Importando el modulo de la siguiente manera:

from odoo.addons.itu_log.models import itu_log

ItuLog = itu_log.ItuLog()

Nos permite enviar mensajes al archivo log configurado en odoo usando el comando:

ItuLog.DebugText(" MENSAJE ")

Antes de enviarse el mensaje comprueba que odoo este en modo debug.

Usage:

For import the module use these commands:

from odoo.addons.itu_log.models import itu_log

ItuLog = itu_log.ItuLog()

Then you can send mensaje to log usin this command:

ItuLog.DebugText(" MENSAGE ")

Before send a mensaje to log, module checks we are in debug mode or not.
