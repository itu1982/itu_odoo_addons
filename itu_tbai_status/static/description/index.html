ITU TBAI STATUS

Displays a column with the TBAI status in the invoice listing

Instalation:

That module requiere this extra modules:
  • account
  • itu_log
  • l10n_es_ticketbai
  • l10n_es_ticketbai_api

Configuration:

Once installed, a new column will appear in the invoice list

Usage:

It works automatically with new invoices.

It contains the following states:

  • Draft: The invoice has not yet been confirmed and therefore has not been sent
  • Pending: Invoice is in queue to be sent
  • Sent: Successfully sent and received
  • Warning: There has been a problem sending the invoice, it is necessary to review the messages received from TBAI
  • Error: A serious error has occurred while sending the invoice or it has been cancelled

Note: Marking the invoice "To review" within the invoice > "other information" recalculates the status

ITU TBAI STATUS

Muestra una columna con el estado de TBAI en el listado de facturas

Instalación:

Este módulo requiere de los módulos:
  • account
  • itu_log
  • l10n_es_ticketbai
  • l10n_es_ticketbai_api.

Configuración:

Una vez instalado, aparecerá una columna nueva en la lista de facturas

Uso:

Funciona automáticamente con las facturas nuevas

Contiene los siguientes estados:

  • Borrador: La factura aún no ha sido confirmada y por tanto no se ha enviado
  • Pendiente: La factura está en cola para ser enviada
  • Enviada: Enviada y recibida correctamente
  • Atención: Ha habido algún contratiempo al enviar la factura, es necesario revisar los mensajes recibidos desde TBAI
  • Error: Se ha producido un error grave al enviar la factura o a sido cancelada

Nota: Marcando la factura "Para revisar" dentro de la factura > "otra información" se recalcula el estado
